# Doom 스타일 궁수 던전 게임

간단한 브라우저 기반 3D 던전 슈팅 게임입니다.

## 실행 방법

1. 터미널에서 프로젝트 루트로 이동합니다.
2. `npm install` (필요한 경우)
3. `npm start`
4. 브라우저에서 `http://localhost:8080`으로 접속합니다.

## 조작법

- WASD: 이동
- 마우스 움직임: 조준
- 좌클릭: 화살 발사
- 좌클릭 길게 누르기: 연속 사격

## 구성

- `index.html` - 게임 실행 페이지
- `style.css` - UI 및 화면 스타일
- `js/game.js` - 게임 로직 및 렌더링
- `server.js` - 간단한 정적 파일 서버
- `Monsters/` - 30종 이상의 몬스터 데이터 파일
- `package.json` - 실행 스크립트
